# Rewire — Daily Mind Reset 🌙

A daily Islamic mind rewiring app. Faith, success, positivity — wired in through 7× daily repetition.

## Features
- 5 daily prayer tracker
- Post-prayer check-ins with AI affirmations
- Morning rewire checklist (dhikr, Quran, gratitude)
- Affirmation page with 3× repetition counter
- Sleep protocol & night reflection
- 7-day streak tracker
